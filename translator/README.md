# 한국어 번역기 (Korean Translator)

한국어를 태국어 또는 벵골어로 번역하는 초등학생 친화적인 웹 번역기입니다.

## 🌟 주요 기능

- **다중 언어 지원**: 한국어 → 태국어 / 벵골어
- **실시간 번역**: MyMemory Translation API 사용
- **초등학생 수준**: 태국어(초3), 벵골어(초4) 수준의 이해하기 쉬운 번역
- **사용자 친화적 UI**: 밝고 직관적인 인터페이스
- **복사 기능**: 번역 결과를 클릭 한 번으로 복사
- **예시 문장**: 자주 사용하는 문장 빠른 입력
- **반응형 디자인**: 모바일, 태블릿, PC 모두 지원

## 🎯 대상 사용자

- 태국어/벵골어를 배우는 초등학생
- 외국어 교사 및 학부모
- 다문화 가정
- 언어 학습자

## 🛠️ 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 그라데이션, 애니메이션, Flexbox
- **JavaScript (ES6+)**: 비동기 API 호출, DOM 조작
- **Translation API**: MyMemory Translation API (무료)

## 📖 사용 가이드

1. **언어 선택**: 태국어(🇹🇭) 또는 벵골어(🇧🇩) 버튼 클릭
2. **텍스트 입력**: 번역할 한국어를 입력창에 입력
3. **번역 실행**: "번역하기" 버튼 클릭 또는 `Ctrl+Enter` (Mac: `Cmd+Enter`)
4. **결과 확인**: 번역된 텍스트가 아래 창에 표시
5. **복사**: "복사하기" 버튼으로 결과를 클립보드에 복사

## 🔧 API 정보

**MyMemory Translation API**
- 제공: MyMemory (Translated.net)
- 비용: 무료
- 제한: 일일 1000회 (일반 사용에 충분)
- API 키: 불필요
- 문서: https://mymemory.translated.net/doc/

## ⚠️ 주의사항

- 인터넷 연결이 필요합니다
- 매우 긴 텍스트는 번역 품질이 낮을 수 있습니다
- 전문 용어나 은어는 정확하지 않을 수 있습니다
- 교육용으로 적합하며, 공식 문서 번역에는 전문가 검토 권장
