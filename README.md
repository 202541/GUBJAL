# GUBJAL - 초등학생 교육 앱 모음집

초등학생을 위한 재미있고 유익한 웹 애플리케이션 모음입니다.

## 📚 프로젝트 목록

### 1. 🎯 구구단 퀴즈 앱 (Multiplication Quiz App)

초등학교 3학년을 위한 재미있는 구구단 학습 웹 애플리케이션

**주요 기능:**
- 다국어 지원 (한국어/태국어)
- 2단~9단 랜덤 문제 생성
- 실시간 점수 및 피드백
- 어린이 친화적 디자인

**접속:** [구구단 퀴즈 실행](https://202541.github.io/GUBJAL/)

**폴더:** `/` (루트)

---

### 2. 🌍 한국어 번역기 (Korean Translator)

한국어를 태국어 또는 벵골어로 번역하는 초등학생용 번역기

**주요 기능:**
- 한국어 → 태국어/벵골어 실시간 번역
- 초등학생 수준의 이해하기 쉬운 인터페이스
- 복사 기능 및 예시 문장 제공
- MyMemory Translation API 사용

**접속:** [번역기 실행](https://202541.github.io/GUBJAL/translator/)

**폴더:** `/translator`

---

## 🛠️ 기술 스택

모든 프로젝트는 다음 기술로 제작되었습니다:
- **HTML5**: 시맨틱 마크업
- **CSS3**: 그라데이션, 애니메이션, 반응형 디자인
- **JavaScript (ES6+)**: 인터랙티브 기능

## 🎯 대상 사용자

- 초등학교 3~4학년 학생
- 외국어를 배우는 어린이
- 다문화 가정
- 초등 교육 종사자 및 학부모

## ✨ 특징

- **초등학생 친화적**: 큰 버튼, 밝은 색상, 쉬운 인터페이스
- **반응형 디자인**: 모바일, 태블릿, PC 모두 지원
- **다국어 지원**: 한국어, 태국어, 벵골어
- **무료**: 완전 무료로 사용 가능
- **설치 불필요**: 웹 브라우저만 있으면 OK
