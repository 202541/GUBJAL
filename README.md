# 구구단 퀴즈 앱 (Multiplication Quiz App)

초등학교 3학년을 위한 재미있는 구구단 학습 웹 애플리케이션입니다.

## 🌟 주요 기능

- **다국어 지원**: 한국어 및 태국어 완벽 지원
- **인터랙티브 퀴즈**: 2단부터 9단까지 랜덤 문제 생성
- **즉각적인 피드백**: 정답/오답에 대한 시각적 피드백
- **점수 추적**: 실시간 점수 및 문제 진행도 표시
- **어린이 친화적 디자인**: 밝고 친근한 색상과 큰 버튼

## 🎯 대상

초등학교 3학년 학생들을 위한 교육용 앱

## 🛠️ 기술 스택

- HTML5
- CSS3 (그라데이션, 애니메이션, 반응형 디자인)
- JavaScript (ES6+)

## 🚀 사용 방법

1. 웹 브라우저에서 `index.html` 파일을 엽니다
2. 원하는 언어(한국어/태국어)를 선택합니다
3. "시작하기" 버튼을 클릭합니다
4. 화면에 표시되는 구구단 문제를 풀어봅니다
5. 10문제를 모두 풀면 최종 결과를 확인할 수 있습니다

## 📁 파일 구조

```
GUBJAL/
├── index.html      # 메인 HTML 파일
├── style.css       # CSS 스타일시트
├── script.js       # JavaScript 로직
└── README.md       # 프로젝트 문서
```

## ✨ 특징

- 중복되지 않는 10개의 랜덤 문제
- 정답 주변의 유사한 숫자로 구성된 오답 선택지
- 애니메이션 효과 (바운스, 페이드인, 정답/오답 피드백)
- 모바일 반응형 디자인
- 성적에 따른 다양한 격려 메시지
